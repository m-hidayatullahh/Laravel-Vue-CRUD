# setup
```bash
git clone https://github.com/m-hidayatullahh/Laravel-Vue-CRUD.git
```

```bash
cd Laravel-Vue-CRUD
```

setup .env
```bash
DB_CONNECTION=mysql
DB_HOST=127.0.0.1
DB_PORT=3306
DB_DATABASE=warga_db
DB_USERNAME=root
DB_PASSWORD=
```

```bash
php artisan migrate
```

```bash
npm install
```

```bash
npm run dev
```

run backend
```bash
php artisan serve
```
